# DigitalOcean 프로모션 코드 (할인 코드) / 2019년 봄
##Jan. 2019 (2019년 1월)

디지털오션 클라우드 서버를 사용하는데, 25달러를 절약하세요.

## 할인 받는 방법은?
#### 1) 여기를 클릭하여 [가입하기](https://m.do.co/c/901875a721b5) 하면, 우선 10달러를 얻을 수 있어요.
#### 2) 지불방법을 선택하고 프로모션 코드  `LOWENDBOX` 를 입력하면, 추가로 15달러를 더 얻을 수 있어요.
